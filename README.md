## Feeshmora-Playground

![fesh](https://cdn.discordapp.com/attachments/884835682202304572/943311519553306664/unknown.png)

Feeshmora Playground: Developed a compact OpenGL engine that allowed for a practical application of essential graphics programming concepts. This project served as a platform for hands-on learning and provided experience in areas like GLSL shaders, phong lighting, frontend support, scalability of both models and states, and even some entity support.
